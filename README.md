# mariessupercoolkeeb

This is my own personally designed keyboard, mariessupercoolkeeb! It's a keyboard with LED underlighting for each key, 70 keys (woah thats a lot), 2 encoders, and an oled screen. The microcontroller is a raspberry pi pico. I made this because I'm a laptop warrior and want an actual keyboard to plug into my laptop, maybe I'll finally get better at Valorant! (no its a skill issue)
